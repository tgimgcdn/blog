## 编写博客
博客文章位于blog目录，您可以将您编写的mdx或者md文件上传到这个目录，如果需要创建分类，也是在blog目录下创建新的文件夹，比如我就在blog目录下面创建了一个tech的文件夹，所有的技术类的文章都放置在这个文件夹里面。
<FileTree>
- src/
  - content/
    - docs/
      - **blog/**
        - tech/
</FileTree>
如果您创建了分类，那么您需要同步在categories文件夹下面创建跟您分类名一致的mdx文件，比如您创建了一个tech的分类，那么就要在categories文件夹下面创建一个tech.mdx。
<FileTree>
- src/
  - content/
    - docs/
      - blog/
      - **categories/**
        - index.mdx
        - **tech.mdx**
</FileTree>

tech.mdx内容如下，如果您创建的其它分类，相应修改内容即可，需要修改的地方我都标注好了。
```md "技术" "tech"
//src/content/docs/categories/tech.mdx
---
title: 技术文章
description: 查看所有技术相关文章
template: splash
---

import CategoryList from '../../../components/CategoryList.astro';

<CategoryList category="tech" /> 
```
如果上述工作您都准备好了，那么恭喜您，您可以开始编写您的第一篇博客文章了，比如您创建了一个life的分类，那么您可以在改文件夹里面创建一个mdx文件，作为您的第一篇博客文章。
<FileTree>
- src/
  - content/
    - docs/
      - blog/
        -**life/**
          -**My first blog.mdx**
      - categories/
        - index.mdx
        - **life.mdx**
</FileTree>
My first blog.mdx的内容如下,必填的选项我都已经高亮标注：
```md "title: " "date: "
//src/content/docs/blog/life/My first blog.mdx
---
title: "我的第一篇博客" 
excerpt: "这是我的第一篇博客，欢迎您的到来！"
giscus: true
date: 2025-05-09
lastUpdated: 2025-05-10
category: life
thumbnail: https://cdn.canjie.org/AgAD4BgAAmga2VQ.webp
tags:
  - "博客"
---
这是我的第一篇博客，欢迎您的到来！
This is my first blog, welcome!
```
备注:
```md "title: " "date: "
//src/content/docs/blog/life/My first blog.mdx
---
title: "我的第一篇博客"                                           //必填项，博客标题，会在您的博客url中出现，例如: https://yourdomain.com/blog/life/我的第一篇博客
excerpt: "这是我的第一篇博客，欢迎您的到来！"                       //选填项，博客摘录，介绍博客的大致内容，支持基本的 Markdown 格式，不需要可以删除该行
giscus: true                                                     //选填项，开启评论，默认所有页面关闭评论，如需开启评论需保留，不需要可以删除该行
date: 2025-05-09                                                 //必填项，博客发布日期
lastUpdated: 2025-05-10                                          //选填项，博客更新日期，如无更新日期可以删除该行
category: life                                                   //选填项，博客分类，如果博客在/src/content/docs/blog文件夹里面，在此处设定博客分类会将博客自动归到life分类，如果博客在/src/content/docs/blog/life文件夹里面，会自动归为life分类
thumbnail: https://cdn.canjie.org/AgAD4BgAAmga2VQ.webp           //选填项，缩略图，可以为url，也可以为图片地址，如果为图片地址的话将文件置于/public文件夹下面，比如文件位于/public/images里面，则此处填写/public/images/My first blog.jpg 其中My first blog为图片名称，需更改为真实的图片名称，如果不填写此项，默认使用/public文件夹下面的social-card.webp文件
tags:                                                            //选填项，标签，每行一个
  - "博客"
  - "Blog"
---
这是我的第一篇博客，欢迎您的到来！                                  //博客正文
This is my first blog, welcome!
```
