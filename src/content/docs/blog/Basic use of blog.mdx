    - HomePage.astro              // 首页组件，展示博客文章列表和分页

    - Page.astro                  // 页面布局组件，定义页面的基本结构
    - PageTitle.astro             // 页面标题组件，处理页面标题的显示

  - content/
    - docs/
      - 404.mdx                   // 404 错误页面，自定义的未找到页面

      - categories/
        - index.mdx               // 分类页面，展示所有文章分类
        - tech.mdx                // 技术文章分类页面
      - index.mdx                 // 网站首页内容
      - link.mdx                  // 外部链接确认页面
      - page/
        - index.mdx               // 分页首页
  - pages/

      - redirect.js               // 重定向 API 处理程序，处理外部链接跳转

  - content.config.ts             // 内容集合配置文件，定义内容的数据结构

## 编写博客
博客文章位于blog目录，您可以将您编写的mdx或者md文件上传到这个目录，如果需要创建分类，也是在blog目录下创建新的文件夹，比如我就在blog目录下面创建了一个tech的文件夹，所有的技术类的文章都放置在这个文件夹里面。
<FileTree>
- src/
  - content/
    - docs/
      - **blog/**
        - tech/
</FileTree>
如果您创建了分类，那么您需要同步在categories文件夹下面创建跟您分类名一致的mdx文件，比如您创建了一个tech的分类，那么就要在categories文件夹下面创建一个tech.mdx
<FileTree>
- src/
  - content/
    - docs/
      - 404.mdx
      - blog/
      - **categories/**
        - index.mdx
        - **tech.mdx**
</FileTree>
tech.mdx内容如下，如果您创建的其它分类，相应修改内容即可，需要修改的地方我都标注好了，
···md
//src/content/docs/categories/tech.mdx
---
title: **技术文章**
description: 查看所有**技术**相关文章
template: splash
---

import CategoryList from '../../../components/CategoryList.astro';

<CategoryList category="**tech**" />
